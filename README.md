# Трассировка автономных систем

## Использование

Для работы необходимо: 
1. ОС с установленной утилитой traceroute;
2. Установить неоходимые пакеты для python из файла requirements.txt

```bash
python3 trace.py address
address - Либо IP адрес, либо доменное имя
```

Если address валидный, то пользователю в консоли выводится таблица:
```bash
 №  |     IP    |   AS   |      Country        |    Provider     |
```
